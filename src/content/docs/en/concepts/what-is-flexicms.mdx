**FlexiCMS** es un **Headless CMS open-source** diseñado para ofrecer el máximo nivel de flexibilidad, control y personalización en la gestión de contenidos. Te permite definir la estructura de tu base de datos, construir un panel de administración a medida y consumir el contenido desde cualquier tecnología o plataforma a través de una **API REST robusta y eficiente**.

## Enfoque y filosofía

FlexiCMS no es solo un CMS. Es una **herramienta para crear CMS y paneles de administración a medida**, adaptados a las necesidades específicas de cada proyecto. Está pensado para desarrolladores que buscan evitar soluciones monolíticas y prefieren un sistema desacoplado, potente y adaptable.

Su enfoque **headless** permite separar completamente el backend del frontend, lo que significa que puedes usar cualquier framework moderno —Astro, Svelte, Angular, React, Vue, Next.js, etc.— para consumir y mostrar tu contenido.

## Colecciones

El núcleo de FlexiCMS gira en torno al concepto de **colecciones**, que definen las entidades o modelos de datos de tu proyecto (por ejemplo: `Users`, `Posts`, `Pages`, `Products`, `Lessons`, etc.).

Cada colección genera automáticamente:

- Una sección en el panel de administración.
- Una API REST con operaciones CRUD.
- Soporte para validaciones, permisos y lógica personalizada mediante hooks.

Esto te permite modelar cualquier tipo de aplicación de forma sencilla y estructurada.

## Fields

Cada colección incluye un conjunto de **fields** que definen los campos de esa entidad. Estos campos no solo estructuran la base de datos, sino que también construyen automáticamente la interfaz del panel de administración.

Los fields pueden ser de múltiples tipos: texto, número, fecha, booleano, relaciones entre colecciones, listas, selecciones, campos condicionales, entre otros.

Cada campo puede configurarse con:

- Validaciones personalizadas
- Visibilidad condicional o por rol
- Hooks específicos por campo
- Valoraciones por defecto o dinámicas

## Panel de administración

El panel de administración se genera automáticamente a partir de tus colecciones y fields. Esto permite:

- Una interfaz clara para gestionar contenido
- Formularios generados automáticamente según tu estructura
- Personalización de vistas por colección o por campo
- Control de accesos según rol

## API REST

FlexiCMS expone una API REST completa y extensible. Por cada colección que definas, dispondrás de:

- Rutas para crear, leer, actualizar y eliminar datos
- Filtros avanzados, ordenación, paginación
- Middleware de autenticación y control de acceso
- Posibilidad de extender el comportamiento con lógica personalizada (hooks)

Puedes consumir la API desde cualquier plataforma: sitios web, aplicaciones móviles, TV, smartwatches, etc.

## Autenticación y control de acceso

Incluye un sistema de autenticación robusto, con:

- Gestión de usuarios y sesiones
- Control de acceso por rol
- Permisos granulares por colección y por campo
- Hooks para definir lógica de seguridad personalizada

## Hooks

FlexiCMS permite insertar lógica personalizada mediante hooks:

- **Hooks por colección**: para modificar, validar o bloquear operaciones como crear, leer o actualizar documentos.
- **Hooks por campo**: para establecer valores por defecto, validaciones dinámicas o transformación de datos.

## Casos de uso

FlexiCMS es ideal para múltiples tipos de proyecto:

- Blogs y sitios de contenido
- Tiendas online (eCommerce)
- Plataformas de e-learning
- Sistemas de reservas o gestión
- Landing pages y microsites
- SaaS multisitio o multitenant

## Multiplataforma y multicanal

Desde un único panel puedes gestionar contenido para:

- Varios sitios web
- Apps móviles
- Aplicaciones de escritorio
- Dispositivos conectados (TV, smartwatch, etc.)

Gracias a su arquitectura desacoplada y su API, puedes centralizar el contenido y distribuirlo donde lo necesites.

## Tecnologías utilizadas

FlexiCMS está desarrollado con tecnologías modernas, priorizando el rendimiento, la escalabilidad y la facilidad de despliegue. Funciona en entornos ligeros, sin necesidad de infraestructura pesada ni dependencias complejas.

## ¿Por qué elegir FlexiCMS?

- Open-source, libre y sin bloqueo de proveedor
- Ligero, modular y fácil de extender
- Personalización total del panel y la base de datos
- Integración sencilla con cualquier frontend
- Ideal para arquitecturas headless modernas
